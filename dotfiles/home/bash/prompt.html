prompt

#

prompt contains various Bash prompts and helper methods I have collected and/or written over the years. Only one can be used at a time, obviously, so only the last entry is used.

These have been tested on Arch Linux using Bash 2.2.24.


    

  

  

  

    

      

        #
      

      
Single-Line Prompts

    

    

      

    

  

  

  

    

      

        #
      

      
This is the default Arch Linux prompt.



    

    

      
PS1='[\u@\h \W]\$ '

    

  

  

  

    

      

        #
      

      
This is a basic prompt with custom colors and the current working directory
displayed.



    

    

      
PS1='\[\e[0;31m\]\u\[\e[m\] \[\e[1;34m\]\w\[\e[m\] \[\e[0;31m\]\$ \[\e[m\]\[\e[0;32m\] '
